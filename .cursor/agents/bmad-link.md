# BMad Agent Link Configuration

BMad is available system-wide via `npx bmad-method`.

## To use BMad in this project:

1. **Via npx (Recommended)**: Use `npx bmad-method <command>` directly
2. **Initialize in project**: Run `npx bmad-method install` and provide project path when prompted

## Quick Commands:
- `npx bmad-method --help` - Show available commands
- `npx bmad-method install` - Install BMad Core in this project
- `npx bmad-method plan` - Use planning agent
- `npx bmad-method code` - Use coding agent
